# Курс по электрическому приводу

Этот репозиторий содержит материалы учебного курса по электрическому приводу: лекции, семинары и практические задания по моделированию и управлению электромеханическими системами.

## Темы лекций

1. **Лекция 1. Введение в электрический привод** — вводная лекция о структуре электропривода, его основных элементах и роли в современных технических системах.
2. **Лекция 2. Двигатели постоянного тока** — рассматриваются принципы работы, характеристики и режимы двигателей постоянного тока.
3. **Лекция 3. Преобразователи в электроприводах постоянного тока** — разбираются силовые преобразователи для ДПТ и их влияние на управление скоростью и моментом.
4. **Лекция 4. Управление электроприводом постоянного тока** — рассматриваются базовые контуры и подходы к управлению током, скоростью и положением ДПТ.
5. **Лекция 5. Синхронные машины** — даются устройство, физические принципы и эксплуатационные особенности синхронных машин.
6. **Лекция 6. Моделирование синхронных машин, PMSM** — посвящена математическому и имитационному моделированию синхронных машин для анализа динамики привода.
7. **Лекция 7. Асинхронные машины** — рассматриваются принципы работы, параметры и типовые режимы асинхронных машин.
8. **Лекция 8. Инверторы и виды модуляции в приводах переменного тока (часть 1)** — вводятся инверторы и основные методы модуляции для управления приводами переменного тока.
