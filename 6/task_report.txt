1 вопрос

Аналитическое решение

На холостом ходу Tem = 0 ⇒ Iq = 0 (при Id = 0).
Предельная частота определяется условием, что напряжения хватает на компенсацию противо-ЭДС:
U ≈ ωe * ψ

Отсюда:
ωe_max = U / ψ  
ωm_max = ωe_max / p  
f_max = ωe_max / (2π)

Подставляя U = 300 В (ограничение в модели), ψ = 0.418 Вб, p = 3:
ωe_max ≈ 718 рад/с  
ωm_max ≈ 239 рад/с  
f_max ≈ 114 Гц  


Моделирование

В модели мы удерживаем Id = Iq = 0 и увеличиваем скорость (а значит и частоту поля);
 момент, когда ток Iq перестает быть нулевым из-за насыщения по напряжению, соответствует тому же условию U = ωeψ, что и в аналитике.

вопрос 2

Ограничения по току у нас фактически нет, поэтому максимальный момент ограничивается лишь напряжением на статорных обмотках.
скрипт task_run2 перебирает разные значения токов Id Iq в большом диапазоне и проверяет, можем ли мы их удержать при доступном напряжении 300В

результат работы скрипта task_run2:
Максимальный допустимый момент найден:
Id_ref = -10.00 A
Iq_ref = 1095.00 A
Id_avg = -9.9999 A
Iq_avg = 1094.9999 A
Torque_avg = 2061.3264 Nm


Двигатель был разогнан до 750 об/мин, после чего при фиксированной скорости перебирались задания токов Id и Iq.
Максимальный режим, в котором регуляторы еще удерживали заданные токи без нарушения ограничения по напряжению, был получен при Id ≈ -10 А и Iq ≈ 1095 А. 
Средний момент на валу составил примерно 2061 Нм. Следовательно, максимальный допустимый момент нагрузки при 750 об/мин равен около 2.06 кН·м.

Проверим найденную точку по MTPA.

Для найденного режима:
Id = -10 А
Iq = 1095 А

Полный ток статора:
Is = sqrt(Id^2 + Iq^2)
Is = sqrt(10^2 + 1095^2) ≈ 1095 А

Для MTPA:
ψ = 0.418 Вб
Ld = 1.12e-3 Гн
Lq = 1.16e-3 Гн
ΔL = Lq - Ld = 0.04e-3 Гн

Id_MTPA = (ψ - sqrt(ψ^2 + 8*ΔL^2*Is^2)) / (4*ΔL) = -12.5 А

Iq_MTPA = sqrt(Is^2 - Id_MTPA^2) = 1095 А

То есть MTPA даёт практически ту же точку:
Id = -12.5 А, Iq ≈ 1095 А.
Разница с результатом перебора из-за дискретности этого самого перебора)

Ld и Lq почти равны, поэтому оптимальное значение Id близко к нулю.
